# yinling-zhikang-agent
TRAE AI创造力大赛初赛作品：银龄智康Agent，面向50-75岁中老年的纯前端轻量化健康AI智能体。基于Vue3+Element Plus开发，内置自主调度AI Agent，实现服药提醒、体检白话解读、体征可视化记录、子女关怀模拟、个性化健康科普五大模块；初赛无后端接口，全部数据通过localStorage本地存储，采用2026磨砂玻璃拟态适老化UI。

1、项目全称：银龄智康 Agent—— 中老年极简自主健康智能体
2、参赛赛道：社会服务赛道・普惠助老方向
3、技术栈：Vite + Vue3 + TS + Element Plus + ECharts + Web Speech API
4、核心创新：前端轻量化自主 AI Agent、适老化玻璃拟态 UI、纯前端无后端 Demo
5、本地启动教程：npm install → npm run dev
