# 智康 Agent · 中老年极简自主健康智能体

> 🏆 TRAE AI 创造力大赛 · 初赛作品（社会服务赛道·普惠助老方向）

面向中老年人的极简纯前端健康管理智能体，全程零后端，所有逻辑前端 JS/TS 实现。

## ✨ 功能亮点

| 模块 | 核心能力 |
| --- | --- |
| 🤖 AI 智能总览首页 | 当日问候 + 待办提醒 + 主动预警推送 + 每周健康小结语音播报 |
| 💊 服药管家 | 新增/冲突检查 + 超时提醒 + 台账 + 方言语音录入 + 药盒图片 |
| 🩺 体检报告 AI 解读 | 模拟上传 + 指标分层解读 + 异常通俗解释 + 调理建议 + ECharts 趋势对比 |
| ❤️ 居家体征看板 | 大字大按钮血压/血糖/心率录入 + 安全阈值预警 + 7/30 天趋势图 |
| 👨‍👩‍👧 子女关怀模拟面板 | 账号绑定流程 + 父母端数据可视化 + 高危异常消息推送弹窗 |
| 📚 AI 精准健康科普 | 根据用户慢病/异常动态推荐 + 大白话 + 语音朗读 + 本地收藏 |

## 🎨 UI/交互风格

- **色彩体系**：柔和医疗蓝(#1677FF)为主，柔和绿色(#52C41A)、醒目红色(#F5222D)、橙色提示(#FAAD14)
- **玻璃拟物**：统一 backdrop-filter blur + rgba(255,255,255,0.7) + 大圆角 20px + 柔和阴影
- **老年无障碍硬性标准**：基础字号 20px+，操作按钮最小宽高 80px，无嵌套弹窗
- **AI 原生体验**：首页 Agent 主动问候 + 预警卡片 + 所有 AI 生成内容标注「AI 智能分析生成」

## 🧠 前端轻量化 AI Agent 能力（纯前端实现，无真实后端）

1. **定时任务**：页面挂载时执行晨间服药提醒、每日健康小结
2. **主动风险识别**：内置血压/血糖安全阈值、药物冲突、空腹用药规则；数据异常自动弹出红色预警
3. **自主内容生成**：体检通俗解读 + 每周健康简报 + 个性化慢病养护方案，由前端规则引擎自动拼接输出
4. **用户状态记忆**：localStorage 保存历史数据，动态匹配专属科普内容，实现个性化服务

## 🧱 技术栈

- **Vite 5** 构建工具
- **Vue 3** <script setup>
- **TypeScript** 严格类型定义
- **Element Plus** 组件库（中文本地化）
- **ECharts 5** 图表
- **原生 Web Speech API** 语音朗读（无需云端）
- **Pinia** 状态管理
- **localStorage** 本地持久化，所有数据本地保存

## 🚀 本地运行

```bash
# 1. 安装依赖（Node.js ≥ 18）
npm install

# 2. 启动开发服务器
npm run dev

# 3. 浏览器访问 http://localhost:5173

# 4. 生产构建
npm run build
```

**完全不需要后端，所有业务逻辑、AI 判断全部由前端实现。下载项目本地打开即可完整体验全部5大核心功能。
